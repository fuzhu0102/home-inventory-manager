# 家庭库存管理 SKILL

自动保存和分析用户输入的库存的 Skill，兼容 openclaw 或 Claude Code 等智能体环境。再也不用怕冰箱里有800年陈年老冻肉了！

## 功能特性

- 录入与修改库存：支持按位置和分类添加、更新物品信息，自动生成备份。
- 删除库存记录：仅在用户明确要求时删除指定记录，确保数据安全。
- 查询库存状态：快速读取库存文件，生成简洁清单。
- 过期提醒：自动检测过期或临期物品，提供提醒信息。
- 数据格式规范：库存数据以 Markdown 表格保存，包含物品、数量、有效期和备注。

## 如何安装这个技能

将本仓库放入技能目录即可使用：

```bash
git clone https://github.com/fuzhu0102/home-inventory-manager ~/.openclaw/skills/home-inventory-manager
```

如果是手动下载压缩包，解压后将目录移动到：

```
~/.openclaw/skills/home-inventory-manager
```

## 在 openclaw 中触发使用

在 openclaw 对话中，您可以通过自然语言描述与家庭库存管理相关的需求来触发该技能。只要需求涉及“录入库存”“查询库存状态”或“检查过期情况”等意图，即可触发该技能。

**推荐触发方式**

- 在需求中明确提到“使用家庭库存管理技能”
- 清晰描述库存操作，例如“添加物品”“删除记录”或“查询库存状态”

**示例**

- 使用家庭库存管理技能，录入卧室床头柜的库存
- 查询卧室床头柜的库存状态
- 删除一楼冰箱冷藏区的过期物品
- 检查所有库存中即将过期的物品

## 你可以让它做什么

- 按位置和分类录入或修改库存记录
- 查询库存状态，生成简洁清单
- 检测过期或临期物品并提供提醒
- 删除指定的库存记录（需明确要求）

## 注意事项

- 所有写操作均通过 `scripts/manage_inventory.py` 脚本完成。
- 删除操作需用户明确要求，且仅删除指定记录。
- 脚本会自动生成备份文件并维护备份数量，无需手动干预。
